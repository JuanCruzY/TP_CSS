Aprendiendo CSS

4.1 Estructura de un archivo CSS

La estructura de un archivo CSS es la siguiente:

identificador {
propiedad: valor;
}

CSS utiliza un estilo de cascada, esto quiere decir que si yo a un identificador le asigno un estilo, y luego unas lineas de código más abajo le asigno otro estilo, el estilo que prevalecerá es el último colocado.

En esta sección no se colocarán estilos para que gráficamente se pueda apreciar cómo pasamos de un archivo HTML sin estilo a uno con estilo.

4.2 Propiedades de Texto en CSS

En CSS usamos la propiedad color para darle color al texto, font-size para darle tamaño a la letra, font-family para indicar la fuente, text-align para la ubicación del texto, text-decoration para subrayar, tachar, etc, letter-spacing para el espaciado entre letra y letra, y line-height para el espaciado entre líneas.
